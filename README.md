# Masterclass Frontend

Starter kit pour suivre la masterclass Frontend disponible gratuitement sur Udemy, lien dans la description du dépôt GitHub.


## Prérequis

Pas de prérequis particulier pour suivre cette masterclass. Il est cependant recommandé de connaître quelques les bases du HTML, CSS.

## Installation

Rien de spécial en dehors d'un éditeur de code (Visual Studio Code, Sublime Text, Atom, etc.) et d'un navigateur web (Chrome, Firefox, Safari, etc.).

## Durée

La masterclass est composée de 3 modules et dure environ 1h20.
